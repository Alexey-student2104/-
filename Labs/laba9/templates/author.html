Описание проекта:

Данный проект представляет собой веб-приложение для отслеживания курсов валют с использованием Flask, SQLAlchemy и API Центробанка России.

Основные функции:

  • Регистрация и авторизация пользователей
  • Просмотр курсов валют в реальном времени
  • Подписка на интересующие валюты
  • Управление пользователями (для администраторов)
  • Управление валютами (для администраторов)
  • CRUD операции для всех сущностей
  • База данных SQLite с сохранением в файл
